# Bubblesort Scheme

Uses mit-scheme


1. Compile file with mit-scheme --load sorter.scm
2. Call the program, by providing it an unsorted list and calling the sort method, using the following:
1 ]=> (sort '(4 3 90 8 1))

Example of running program:
mit-scheme --load sorter.scm
1 ]=> (sort '(4 3 90 8 1))

Output would be:
;Value 13: (1 3 4 8 90)
